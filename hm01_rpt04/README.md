10.네이버로이동
11.계산기
12.현재시간(누르는초가 홀수,짝수일경우 배경색 다르게)
13.깜짝퀴즈
14.학교홈페이지로 이동
15.수강신청으로 이동
16.사진첩(클릭하면 다음사진으로 넘어가는)
17.풋터에 화면에 관한 정보 추가
